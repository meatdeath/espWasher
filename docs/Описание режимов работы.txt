После выбора режима работы, конфинурации и нажатия на кнопку Старт, машина выполнит следующие действия
Подать сигнал на термотаблетку замка 
Проверить наличие сигнала о закрытии люка
Если конфигурация содержит замачивание, то
{
    Налив холодной воды через клапан №1: 1мин
    Налив холодной воды с помешиванием белья : пока не сработает датчик уровня воды или авария по таймауту 5мин
        Пока ждем постоянно повторяем
        [
            3сек крутим со скоростью барабана 1/2 об/мин для бережной стирки или 1 об/мин для остальных режимов, 
            5 сек стоим
            меняем направление вращение барабана
        ]
    Стираем в холожной воде : пока не истечет таймаут заданный конфигурацией
        Пока ждем постоянно повторяем
        [
            45сек крутим со скоростью барабана 1/2 об/мин для бережной стирки или 1 об/мин для остальных режимов, 
            15 сек стоим
            меняем направление вращение барабана
        ]
    Слив воды, включаем насос : ждем пока не станет воды в баке или авария по таймауту 3мин
    Укладка : 1 мин
        Пока ждем постоянно повторяем
        [
            3сек стоим
            меняем направление вращение барабана
            7сек крутим барабан на скорости 1/2 об/мин для бережной стирки или 1 об/мин для остальных режимов
        ]
    Отжим слегка
        30сек плавно набираем обороты до 100об/мин для бережного режима, до 200 об/мин для остальных режимов : если вибрация сильная идем на укладку максимум 3 раза иначе ошибка
        Держим обороты 30сек
        Выключаем мотор, ждем пока остановится барабан и выключаем насос, когда не станет воды в баке
    }
}

            меняем направление вращение барабана

            45сек крутим со скоростью барабана 1/2 об/мин для бережной стирки или, 2об/мин - для интенсивной стирки и 1 об/мин для остальных режимов,

    Укладка : 1 мин
        Пока ждем постоянно повторяем
        [
            3сек стоим
            меняем направление вращение барабана
            7сек крутим барабан на скорости 1/2 об/мин для бережной стирки или 1 об/мин для остальных режимов
        ]
    Отжим 
        30сек плавно набираем обороты до 100об/мин для бережного режима, до 200 об/мин для остальных режимов : если вибрация сильная идем на укладку максимум 3 раза иначе ошибка
        Держим обороты 30сек
        10сек плавно набираем обороты до заданных в конфигурации
        Держим обороты 3мин
        Выключаем мотор, ждем пока остановится барабан и выключаем насос, когда не станет воды в баке
    }
